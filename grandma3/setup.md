# OSC GrandMA3
## Purpose
This software allows user to do pre-programming and create lighting design for efficient and effective control of lighting setups. 
## Setup of software
<p>1. Download software [grandMA3 onPC Software for Windows, 2.4.2.2](https://www.malighting.com/downloads/products/grandma3/)</p>
<p>2. Once downloaded, click Accept and download</p>
<p>3. Extract the software's zip file </p>
<p>4. Click on 'ma' in file</p>
<p>5. Click on 'grandMA3_onPC_win_v2.4.2.2' in file</> 
<p>6. Click 'Yes'</p>
<p>7. Click "Next" to continue.</p>

<p>8. Click "I Agree"</p>
